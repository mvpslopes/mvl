# Comandos para o Servidor (Hostinger)

## Via SSH:

```bash
cd public_html/api
# ou
cd api

composer require google/analytics-data
```

## Via File Manager (Hostinger):

1. Acesse o File Manager no painel da Hostinger
2. Navegue até a pasta `api/`
3. Se o Composer não estiver disponível, veja a opção abaixo

## Alternativa: Instalar localmente e fazer upload

No seu computador (Windows):

```powershell
cd C:\projetos\SiteMVL\api
composer require google/analytics-data
```

Depois, faça upload da pasta `vendor/` para `api/vendor/` no servidor.
